Welcome to the landing page

This is a blank landing page which inherits from landing_page/base.html. Feel free to add more routes to this app and edit the base.html file to add more content to the landing page.

Editing Instructions

  1. The default favicon can be found in static/landing_page/images/favicon.png. To use your own logo, simply replace favicon.png with your own image (use the same image name or edit base.html to reflect the new name).
  2. There are two CSS files in this app - reset.css and index.css.
    • reset.css is used to reset the default CSS of the browser. I used this CSS reset from Meyerweb but feel free to replace it with Normalize CSS or any other CSS reset.
    • index.css is used to style this page. Feel free to delete this CSS file and replace it with your own.
